# Nurse Shift Scheduling Example

## Usage

```console
$ clinguin client-server --domain-files encoding/nsp.lp instances/nsp-n5-d7-r0.10-rp0-fp0.lp --ui-files ui.lp --custom-classes custom_backend.py --backend CustomBackend --explicit-show
```
